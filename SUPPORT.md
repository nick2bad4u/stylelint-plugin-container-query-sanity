# Support

Thanks for using `stylelint-plugin-docusaurus`.

## How to get help

1. Check project docs first:
   - [README.md](./README.md)
   - [docs/rules/overview.md](./docs/rules/overview.md)
2. Search existing issues:
   - <https://github.com/Nick2bad4u/stylelint-plugin-docusaurus/issues>
3. Open a new issue if needed:
   - <https://github.com/Nick2bad4u/stylelint-plugin-docusaurus/issues/new>

## What to include in an issue

Please include:

- Node.js version
- Stylelint version
- `stylelint-plugin-docusaurus` version
- minimal reproduction stylesheet/config
- expected behavior and actual behavior

If possible, include your Stylelint config snippet (`stylelint.config.*`) that reproduces the issue.

## Support scope

This repository supports:

- bug reports in plugin rules/configuration
- false positives and false negatives in rules
- shareable-config and docs improvements
- release and packaging issues for this package

This repository does not provide private support channels.

## Security reports

For security concerns, do not open a public issue.
Follow [SECURITY.md](./SECURITY.md).

## Community standards
